# taller-informatico-scrum
Este proyecto gestiona las reparaciones del taller “Tu informático te ayuda”
aplicando la metodología SCRUM.

## Historias de Usuario

- Como recepcionista, quiero registrar la entrada de un equipo para controlar las reparaciones.
- Como técnico, quiero registrar el diagnóstico del equipo.
- Como administrador, quiero gestionar proveedores y piezas.
- Como administrador, quiero crear y enviar presupuestos a los clientes.
- Como técnico, quiero registrar el avance de la reparación.
- Como administrador, quiero gestionar la garantía.
